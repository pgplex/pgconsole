# pgconsole

This repository is used exclusively for tracking **bug reports** and **feature requests** for [pgconsole](https://www.pgconsole.com).

- **Bug reports**: Please include steps to reproduce, expected behavior, and actual behavior.
- **Feature requests**: Describe the use case and why it would be helpful.

For billing questions, reach out to [billing@pgconsole.com](mailto:billing@pgconsole.com).
